Getting Started

This guide will show you how to interact with several of Spotify's API endpoints in order to retrieve data from their catalog. At the end, we'll put some code together to make a related playlist based on a searched artist! The guide requires a basic knowledge of HTML/CSS, JavaScript, and JSON. Let's get started!
